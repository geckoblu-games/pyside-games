## Sokoban

![screenshot](./assets/screenshot-sokoban.png)

**Controls**

  - Arrow keys -> Move player
  - U -> Undo last move
  - R -> Restart level
  - ESC -> Choose level
  - n,p -> Next,Prev level

**Credits**

Graphics are modified version of [Kenney's](https://kenney.nl/assets/sokoban) ones
